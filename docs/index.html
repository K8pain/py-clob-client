Documentation Hub

HLD + LLM docs con look editorial, moderno y realmente legible.

Este mini sitio estático convierte la documentación del repositorio en una experiencia visual más amable. La idea es que un humano pueda entender rápido el producto, y que un LLM tenga una guía práctica y accionable para contribuir sin ruido.

⚡ lectura rápida 🧠 contexto para humanos 🪄 diseño editorial moderno

Qué incluye

HLDVisión de producto, arquitectura, riesgos, plan y extensiones futuras.
LLM GuideReglas concretas para que asistentes hagan cambios pequeños, testeables y seguros.

Formato

HTML estático, responsive, con navegación interna y estilo visual consistente. No necesita build step.

Documentos principales

Dos vistas complementarias: una para entender el sistema desde arriba y otra para guiar contribuciones operativas de asistentes o equipos técnicos.

System view

HLD · High Level Design

Explica qué es el SDK, para quién es, cómo funciona, qué riesgos tiene, cómo priorizar trabajo y cuáles son sus extensiones futuras.

 Contributor view

LLM · Working document

Define cómo debería razonar un asistente sobre el repo: estructura, testing, seguridad, edge cases, DoD y ripple effects.

Por qué esta versión es mejor

La iteración anterior cumplía como texto, pero no como artefacto realmente agradable de leer. Esta entrega prioriza claridad visual, jerarquía, navegación y memorabilidad.

Más humana

Se lee como una landing/documentation page y no como un bloque plano de Markdown.

Más útil

Cada sección tiene propósito claro, scannability y navegación ancla.

Más mantenible

Todo vive en archivos HTML estáticos sin dependencia de framework, build o assets externos.

Uso recomendado

Abrir `docs/index.html` en navegador y navegar desde allí. Para compartir, basta con servir la carpeta `docs/` como contenido estático.

Sugerencia: si más adelante quieren publicarlo en GitHub Pages, este formato ya está listo para eso.
Closing note Diseñado para ser visualmente agradable sin sacrificar densidad de información ni mantenibilidad.
